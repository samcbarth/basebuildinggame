# Roadmap

## Prototype

- Build the first playable survival loop.
- Confirm economy, defense, upgrades, and wave pressure work together.
- Tune costs and enemy scaling after playtesting.

## Vertical Slice

- Add visible worker behavior.
- Add selectable buildings and unit command feedback.
- Add at least one additional enemy type.
- Add low-res sprite assets or generated pixel art.
- Add sound placeholders for build, shoot, hit, wave, and game over.

## Alpha

- Add save/load.
- Add expanded tech tree.
- Add multiple maps or terrain blockers.
- Add clearer military unit roles.
- Add build/export process.
