# Asset Pipeline

## Current State

The prototype uses code-drawn placeholder shapes. No external runtime assets are required yet.

## Folder Rules

- `assets/sprites`: final or placeholder sprite sheets.
- `assets/animations`: animation source or exported frames.
- `assets/tilesets`: tile maps and terrain sprites.
- `assets/audio`: sound effects.
- `assets/music`: music loops.
- `assets/fonts`: bitmap or TTF fonts.
- `assets/shaders`: shader experiments.

## Placeholder Rule

When art assets are added, placeholders should be named with a `placeholder_` prefix until replaced.
