# Art Style Guide

## Direction

Low-resolution, low-detail battlefield shapes. Think readable board-game pieces before polished art.

## Current Rules

- Keep silhouettes simple.
- Prefer high-contrast color blocks.
- Avoid detailed textures until the core loop is fun.
- Keep buildings readable at 16x16 grid scale.
- Use placeholder shapes intentionally and replace them later with matching low-res sprites.

## Current Palette Notes

- Blue: HQ and player military identity.
- Gold: scrap/economy.
- Green: construction/defense.
- Red: enemies and danger.
- Purple: barracks/production.
